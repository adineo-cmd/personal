# Free Software & Tools

You can build a fully functional computer setup using only open-source and free software. Here are the essential tools.

## Office Suite

- **LibreOffice:** A powerful free alternative to Microsoft Office (Writer, Calc, Impress).

## Media Tools

- **VLC Media Player:** Plays almost any multimedia file format.
- **OBS Studio:** Free and open-source software for video recording and live streaming.

## Security

- **VeraCrypt:** Disk encryption software for securing sensitive data.

## Utilities

- **7-Zip:** High compression ratio file archiver.
- **Rufus:** Utility to format and create bootable USB drives.
- **Ventoy:** Create a bootable USB drive with multiple ISO files without formatting.
